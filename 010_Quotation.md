# \<q>
----------

\<q>는 짧은 인용을 정리한다

주로 큰따옴표를 삽입한다.

<p><q>인용구</q> 이다.

# \<blockquote>
--------------

\<blockquote>은 긴 인용구를 정리한다.

디스플레이에는 들여쓰기를 한것처럼 나타난다

<blockquote>인용구가

매우 길때

사용</blockquote> 한다
