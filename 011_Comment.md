# HTML Comment Tag
----------------------

\<!-- 주석 -->

형식으로 주석을 추가하는것이 가능하다

여는태그에는 느낌표가 있지만 닫는태그에는 없다

주석은 브라우저엔 표시되지 않지만 코드를 문서화하는데 도움이 된다.

```
<!-- This is a comment -->

<p>This is a paragraph.</p>

<!-- Remember to add more information here -->
```

위의 내용처럼 한번에 한줄씩 주석을 달아서 디버깅하면 에러 위치를 찾아내기 수월하다.
