# HTML 편집기
-------------------

전문 HTML 편집기를 사용하여 웹 페이지를 작성할 수 있지만

HTML을 학습할땐 메모장(PC)이나 TextEdit(MAC)같은 같단한 편집기를 사용하는 것이 좋다

메모장을 열고
```
<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>

<p>My first paragraph.</p>

</body>
</html>
```

을 복사한다음 index.htm 으로 지정하고 인코딩을 UTF-8로 설정하면 브라우저에서 볼 수 있다
