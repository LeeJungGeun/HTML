# multimedia
-----------------

웹 페이지에는 다양햔 형식의 멀티미디어 요소가 포함되어있다

오디오,비디오같은 멀티미디어 요소는 미디어 파일에 저장된다

파일유형은 확장자를 보면 바로 알 수 있다
